# Classification Datasets

This directory contains datasets suitable for classification tasks in machine learning.

## Available Datasets

### iris.csv

**Description:** The famous Iris flower dataset, originally introduced by Ronald Fisher in 1936. This is one of the most widely used datasets for machine learning classification tasks.

**Features:**
- `sepal_length`: Sepal length in cm
- `sepal_width`: Sepal width in cm
- `petal_length`: Petal length in cm
- `petal_width`: Petal width in cm

**Target:**
- `species`: The species of Iris flower (setosa, versicolor, virginica)

**Statistics:**
- Number of samples: 150
- Number of features: 4
- Number of classes: 3
- Samples per class: 50

**License:** Public Domain

**Source:** UCI Machine Learning Repository
